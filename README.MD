# Parent

Parent for all open source Maven projects.

## Usage

Add this parent to your pom.xml:

```xml
    <parent>
        <groupId>com.madden-abbott</groupId>
        <artifactId>parent</artifactId>
        <version>${parent.version}</version>
    </parent>
```
